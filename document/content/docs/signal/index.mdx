---
title: DistributeReadWriteLock
description: DistributeReadWriteLockImp 是一个基于 MongoDB 实现的分布式读写锁工具类，旨在为分布式系统提供线程安全的读写锁机制。该类允许多个节点在共享资源上进行协调，确保在高并发环境下的数据一致性和操作的原子性。
---

## 特性
- 基于 MongoDB 的锁机制：利用 MongoDB 的原子操作（如 findAndModify）实现分布式锁，无需引入额外的中间件。
- 支持读写锁：实现了读写锁的语义，允许多个读操作并发执行，同时确保写操作的独占性。
- 可重入性：同一线程可以多次获取同一锁，避免死锁的风险。
- 锁超时机制：支持设置锁的超时时间，防止因异常导致的死锁。
- 灵活的锁粒度：可以根据业务需求设置不同的锁粒度（如按资源 ID 锁定）。
